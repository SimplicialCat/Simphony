# 19-EDO 音阶表生成脚本

本目录包含用于生成19平均律（19-EDO）相关数据表的Python脚本。

## 脚本说明

### 1. `generate_19edo_table.py`
生成基础的19-EDO音阶表，包含：
- 步数（0-18）
- 音名（使用LaTeX格式）
- 频率比（相对于C4）
- 音分值

**使用方法：**
```bash
python generate_19edo_table.py
```

**输出示例：**
```
| 步数 | 音名 | 频率比 | 音分 |
|------|------|--------|------|
| 0 | $\text{C}$ | $1.000000$ | $0.00$ |
| 1 | $\sharp\text{C}$ | $1.037155$ | $63.16$ |
...
```

### 2. `generate_19edo_detailed.py`
生成详细的19-EDO音阶表，额外包含：
- 键型（白键/紫键/蓝键）
- 间隔类型（大间隔/小间隔）
- 与12-EDO的对比

**使用方法：**
```bash
python generate_19edo_detailed.py
```

**包含表格：**
1. 基础19-EDO表
2. 详细表（含键型和间隔）
3. 19-EDO与12-EDO对比表

### 3. `generate_19edo_scales.py`
生成各调式在19-EDO中的大调音阶：
- 音级与唱名
- 绝对位置
- 频率比和音分
- 与12-EDO的对比

**使用方法：**
```bash
python generate_19edo_scales.py
```

**包含调式：**
- C 大调
- Eb 大调（测试用例中的调式）
- F 大调
- G 大调
- Bb 大调

## 19-EDO 基础知识

### 键盘布局
19-EDO钢琴键盘遵循以下模式：
```
W, P, B, W, P, B, W, P, W, P, B, W, P, B, W, P, B, W, P
```
其中：
- W = 白键（自然音）
- P = 紫键
- B = 蓝键

### 间隔类型
- **大间隔**：紫+蓝双键（位置：1-2, 4-5, 9-10, 12-13, 15-16）
- **小间隔**：单紫键（位置：7, 18）

### 数学原理
- 每个八度分为19个等距音程
- 每步音分 = 1200/19 ≈ 63.16¢
- 频率比 = 2^(n/19)，其中n为步数

### 调式位置
各调式在19-EDO中的主音位置（相对于C）：

| 调式 | 位置 | 调式 | 位置 |
|------|------|------|------|
| C | 0 | F# | 9 |
| C# | 1 | Gb | 10 |
| Db | 2 | G | 11 |
| D | 3 | G# | 12 |
| D# | 4 | Ab | 13 |
| Eb | 5 | A | 14 |
| E | 6 | A# | 15 |
| F | 8 | Bb | 16 |
| | | B | 17 |

## 与12-EDO的主要差异

1. **小三度更窄**：19-EDO的小三度比12-EDO更窄（约315.79¢ vs 300¢）
2. **大三度更宽**：19-EDO的大三度比12-EDO更宽（约378.95¢ vs 400¢）
3. **纯五度略窄**：19-EDO的纯五度比12-EDO略窄（约694.74¢ vs 700¢）

这些差异使得19-EDO能够提供不同的和声色彩，特别适合探索微音程音乐。

## 注意事项

- 所有脚本使用UTF-8编码以正确显示特殊符号

- 音名使用LaTeX格式，适合在支持LaTeX的环境中渲染
